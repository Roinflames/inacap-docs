# Objetivo General
Que el estudiante sea capaz de implementar, optimizar y desplegar un modelo de Deep Learning utilizando herramientas modernas de MLOps y despliegue en distintos entornos (local, nube y edge).

# 1. Contenidos Clave
## 1.1 Optimización de Modelos

Cuantificación (INT8, FP16)

Pruning / poda

Compresión de modelos

Exportación de modelos: SavedModel / H5 / TFLite / ONNX

## 1.2 Despliegue del Modelo

Despliegue local

Despliegue en la nube (AWS, GCP, Azure)

CI/CD aplicado a ML

Edge computing (dispositivos móviles, Raspberry Pi)

## 1.3 Herramientas Importantes

TensorFlow Lite (TFLite)

ONNX + ejecución en GPU

AWS SageMaker / GCP Vertex AI

FastAPI + Docker para servir modelos

MLOps con TFX (TensorFlow Extended)

# 2. Ejemplos Prácticos Recomendados
## 2.1 Deploy en dispositivos móviles

TFLite + MobileNet

Implementación en Android Studio / iOS

## 2.2 Deploy en Cloud

AWS SageMaker (endpoint de inferencia)

Alternativa gratuita: FastAPI + Docker + Render / Railway

## 2.3 Conversión a ONNX

Exportar un modelo CNN a ONNX

Ejecutar inferencia acelerada en GPU (CUDA / DirectML)

# 3. Recursos Complementarios

MLOps con TensorFlow Extended (TFX)

Ejemplo de API de inferencia con FastAPI

Pipeline Dockerizado

Buenas prácticas de escalabilidad y versionamiento de modelos

# 4. Evaluaciones Sugeridas
## ✔ Evaluación Diagnóstica

Clasificación de problemas según tipo de aprendizaje:

Supervisado / No Supervisado / Regresión / Clasificación
Usar problemas sencillos de Kaggle.

## Evaluación Práctica 1

Preprocesamiento de Dataset (Titanic / Iris)
Incluye:

Limpieza

Normalización

División train/test

Visualización básica (matplotlib)

## Evaluación Práctica 2

Entrenar una CNN simple:

Clasificación de imágenes (MNIST, CIFAR-10 o Cats vs Dogs)

Ajuste de hiperparámetros (epochs, learning rate, batch size)

## Evaluación Final

Despliegue de un modelo en alguna modalidad:

Local: API FastAPI + servidor local

Cloud: Render, Railway, AWS SageMaker, GCP Vertex AI

Edge: conversión a TFLite y uso en Android
